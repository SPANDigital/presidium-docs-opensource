---
title: Prerequisites

---

The following tools are required to use Presidium:

1. Install Homebrew

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```

2. Install Presidium
   
    ```bash
    brew install SPANDigital/tap/presidium
    ```
